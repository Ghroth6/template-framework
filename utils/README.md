# Utils模块说明

## 1. 模块概述

### 1.1 功能描述

Utils模块是TFW框架的工具模块，封装了平台相关操作和通用工具函数。该模块为其他模块提供基础工具支持，包括日志、内存管理、线程、文件操作等功能。

### 1.2 模块位置

模块位于项目的`utils/`目录下。
